AI-powered Crypto Twitter (X) & Discord Sentiment Scraper. Web3 projects are obsessed with community sentiment; this tool scrapes real-time mentions and analyzes the "vibe" of any token or project.

## Features
- **Multi-Platform Scraping**: Automated Playwright scraper for X (Twitter) and Web News.
- **Web3 Sentiment Analysis**: AI-driven bullish/bearish scoring and community vibe checks.
- **Competitor Intel**: Compare your project's sentiment against rivals in the space.
- **Strategic Roadmap**: AI-generated marketing and community engagement tasks based on real data.

## Tech Stack
- **Core**: Python, FastAPI
- **Scraper**: Playwright (Headless/Persistent Sessions)
- **AI**: Google Gemini 2.5 Flash
- **Frontend**: Modern Glassmorphism UI (FastAPI Static)

## Getting Started
1. `pip install -r requirements.txt`
2. `playwright install chromium`
3. Set `GEMINI_API_KEY` in `.env`
4. `python main.py`
